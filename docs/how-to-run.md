# HOW TO RUN

---

1. 의존성 패키지 설치

```bash
# use npm
npm i

# use yarn
yarn
```

2. 프로젝트 실행

- 개발자 모드 실행

```bash
# use npm
npm run dev

# use yarn
yarn dev
```

- 프로젝트 빌드

```bash
# use npm
npm run build

# use yarn
yarn build
```

- 프로덕션 모드 실행

```bash
# use npm
npm run start

# use yarn
yarn start
```

- 린트 검사

```bash
# use npm
npm run lint

# use yarn
yarn lint
```

- 스토리북 실행

```bash
# use npm
npm run storybook

# use yarn
yarn storybook
```

- 스토리북 빌드

```bash
# use npm
npm run build-storybook

# use yarn
yarn build-storybook
```

- 테스트 실행

```bash
# use npm
npm run test

# use yarn
yarn test
```
